# 🎵 Music Deck Spinner

A simple, lightweight companion app for music card party games.

Spin the wheel to randomly select which deck to draw from — no more arguments at the table.

> ⚠️ This is an unofficial fan-made companion app and is not affiliated with or endorsed by any game publisher.

---

## ✨ Features

- 🎡 Smooth animated spinning wheel
- 🎯 Random deck selection
- 🎨 Pixel-style UI
- 🌍 Multi-language support (EN / DE)
- 💾 Saves selected decks between sessions
- 📱 Built with Expo + React Native

## 🛠 Tech Stack

- React Native (Expo)
- Reanimated 3
- React Native Gesture Handler
- NativeWind (Tailwind for RN)
- i18next (translations)
- AsyncStorage (persistence)

---

## 🚀 Development

### Install dependencies

```bash
npm install
```

### Start Dev Server

```bash
npx expo start
```

### Run on Android

```bash
npx expo run:android
```

## 🚀 Build Project with Expo

```bash
npx expo prebuild
```
